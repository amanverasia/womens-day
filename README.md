# Women's Day Microsite

Static Next.js App Router microsite for International Women's Day.

## Local Development

```bash
npm i
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel (Git Import)

1. Push this project to a GitHub repository.
2. Go to Vercel and click **Add New Project**.
3. Import the GitHub repository.
4. Keep default Next.js settings and deploy.

## Optional Vercel CLI

```bash
vercel
vercel --prod
```
